# 证券板块行情启动信号回测系统 — 设计文档

## 1. 目标

从历史证券板块行情中，通过系统性回测找出有效的技术指标启动信号，帮助捕捉下一轮证券板块行情。核心要求：**不过早买入、不错过行情**，两者平衡。

## 2. 数据来源

- **API**：双数据源架构 — AKShare 主、tushare 备。AKShare 失败时自动回退 tushare，成分股列表额外有硬编码兜底。详见 [双数据源改造设计](2026-06-17-双数据源改造-design.md)。
- **时间范围**：2010-01-01 至今（数据从 2008 年开始拉取，2 年预热期确保 250 日均线等长周期指标有效）
- **周期**：日线 + 周线
- **板块指数**：中证证券公司指数（399975.SZ）
- **个股**：中证证券公司指数全量成分股（当前 49 只），通过 `ak.index_stock_cons_csindex("399975")` 动态获取。早期设计曾用 10 只候选（中信证券/海通证券/华泰证券/广发证券/招商证券/光大证券/中信建投/中泰证券/中银证券/财达证券），现已扩展到全成分股。
- **缓存**：数据拉取后存 `data/` 目录 CSV。工作日 15:00 后首次加载自动触发增量拉取当天收盘数据（`_needs_eod_update`）。

## 4. 行情周期识别

检测 2010 年至今的板块指数牛市周期。经人工审阅后确认 9 轮行情（2010年后），含修订：
- 合并 2014-07~2015-01 与 2015-02~2015-05 为同一轮（2014-2015大牛市）
- 新增 2012-01~2012-06（+50%/148天，被原算法遗漏）

包含 2009 年行情作为参考锚点。结果：10 行（含 2009），9 轮用于 2010+ 回测。

## 5. 技术指标

所有指标使用 TA-Lib 标准库计算（RSI 用 Wilder 平滑、ADX/CCI/布林带等均为标准实现）。

### 主要指标
| 指标 | 类型 | 信号定义 |
|------|------|---------|
| CCI 脱离超卖 | 动量 | CCI 从 -100 以下上穿 -100 |
| MACD 柱线缩短 | 趋势 | 绿柱连续 3 天收窄 |
| MACD 底背离 | 趋势 | 相邻波段低点：价格新低、DIF 未新低 |
| OBV 底背离 | 成交量 | 价格下行、OBV 上行 |
| RSI 底背离 | 动量 | 价格新低、RSI 未新低 |
| 均线回归 | 趋势 | 长期低于均线后首次站回 |
| ADX 拐头 | 趋势 | ADX 低位拐头 + +DI 上穿 -DI |
| KDJ J 值反转 | 动量 | J 值 <0 后上穿 +20 |
| 布林带收窄 | 波动 | 带宽创 N 日新低 |
| 布林带下轨反弹 | 波动 | 价格沿下轨运行后站上中轨 |

## 6. 语境过滤（filter_signals）

信号触发后需满足以下条件才计入有效：

1. **AND 条件**：价格低于 250 日均线 **且** 从上一轮行情高点下跌超过 20%
2. **状态锁存**：条件满足后保持有效 5 个交易日。解决"市场先触底再反弹到启动"期间信号被误过滤的问题（如2025年4月行情：条件在04-07满足，CCI在04-11才触发）
3. **冷却期**：同一指标两次信号至少间隔 15 个交易日
4. **参考高点**：取最近一轮已完成行情的结束价

总计约 56 个信号规则参与回测（含不同参数组合）。

## 7. 回测与评分

### 信号判定
行情启动日 T，信号触发日 S：若 S ∈ [T-45, T]，命中；S ∈ (T, T+10]，太晚；否则错过。

### 综合得分
```
得分 = 命中率 × 0.5 + 信号有效率 × 0.2 - |平均提前天 - 5| / 30 × 0.3
```

## 8. 当前最优结果

| 排名 | 指标 | 命中 | 总信号 | 有效率 | 得分 |

## 9. Streamlit 界面

### 4 个标签页
1. **行情周期总览**：板块指数 K 线图 + 行情周期标注
2. **指标回测排行榜**：可排序表格 + 筛选 + 指标详情（每轮行情 K 线 + 指标图 + 信号标记）+ 热力图
3. **个股回测对比**：各股票 × 指标得分表 + 热力图 + 个股行情信号详情
4. **行情跟踪**：指数 + 个股近 1 年 K 线图（含 CCI + MACD 实时信号 + 年线 + 跌 15% 线 + 参考高点）

## 10. 技术栈

- Python 3.10+, TA-Lib, tushare, pandas, numpy, scipy, plotly, streamlit

## 11. 约束与边界

- 不做实盘交易接口，仅做信号分析
- 不预测行情高度，只判断启动时机
- 仅用技术指标，不涉及基本面
- 历史回测结果不代表未来表现
